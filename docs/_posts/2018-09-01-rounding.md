## N2D(u, v) := (u, v) -> x
```python
B_m1 = 1. / B
B2 = B * B
B_2 = B / 2


t1 = u
t2 = B_m1 * v
x = t1 + t2
```

## D2N(x) := x -> (u, v)
```python
t1 = x * B2
t2 = t1 + B_2

a1 = floor(t2 / B2) # t2 // B2
r1 = t2 - a1 * B2
a2 = floor(r1 / B) # r1 // B

u = a1
v = a2
```
